# All Days Calendars

This repository contains a number of scripts to generate ICS calendars with all-day events of interest.
This targets the French language.
GitHub workflows are run at regular intervals to keeps the calendars updated.

+ [fetedujour.ics][fetedujour] contains the name to celebrate today in France.
  Results use the [API](https://fetedujour.fr/api) from [fetedujour](https://fetedujour.fr/).
+ [histoire.ics][histoire] contains one historic event at random, in French.
  Results are scraped from [calagenda](www.calagenda.fr)

  [fetedujour]: https://efrecon.github.io/allday.ics/fr-FR/fetedujour.ics
  [histoire]: https://efrecon.github.io/allday.ics/fr-FR/histoire.ics
